# Device Health Metrics Definitions

## Battery Metrics
- Voltage, temperature, charge cycles thresholds.
- Policy Bus alert triggers for abnormal discharge.

## Thermal Metrics
- CPU/GPU temperature bands and mitigation actions.
- UI representation using neon warning triangles.

## Hardware Integrity
- Integrity checks for storage, sensors, and secure elements.
- Logging strategy referencing neon padlock iconography.
