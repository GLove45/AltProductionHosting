# Zion Core

Foundation layer responsible for verified boot, kernel hardening, device driver management, and SELinux policy enforcement.

Planned deliverables:
- Hardened kernel configuration for Xperia 10 IV (pdx224).
- Verified boot chain configuration and vbmeta generation scripts.
- Device bring-up notes for power, NFC, fingerprint, and camera subsystems.
- SELinux policy overlays for core domains.
